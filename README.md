# DeevDialog

This is simple Dialog android library .


## Follow this init

### Add your project level gradle
 `allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}`

and 
  
### Add your app level gradle

`dependencies {
	        compile 'com.github.HeinXtet:DeevDialog:0.1.0'
	}`
